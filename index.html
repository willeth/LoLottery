Low Odds Lottery

A Simple, Transparent and Fast Game

Contract Address

We believe that playing a lottery should be simple, transparent and fun. Here at Lolottery, there is no complicated odds system, unknown number of players or variable ticket price. The game is run on a publically available ethereum smart contract so no tampering can occur, the ramdomness for the winner selection is generated off chain using the "" library removing the ability for miners to rig the game. The game is simple, you buy your ticket and once the game is full the smart automatically picks a winner and pays out the prize pool to their address. All players addresses are listed on the website and the winner is listed on the website along with the transaction hash.
